About

This project was the final project for one of my third year classes. We took the already completed Java Maze Application and porten it to Android. This was by far one of the most challenging but rewarding projects that I have ever done. We learned the entire android application layout and cerated this application in around 2 months, which felt almost too fast at some points. However, the app features a custom created view to display the maze game as well as multithreading to prevent UI hanging.
