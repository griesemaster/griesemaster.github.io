About

This project was the final project for one of my third year classes. The image that is opened in the clip is entirely generated from the script, it is a PPM image in which each pixel is a RGB value. There is a list of environment shapes passed to the script and the image is generated. It is only capable of spheres, but other geometric figures could be added, with intense additional work.

Raytracer

This project was developed in Ubuntu and complied and run from BASH shell

  • Language: C
  • Source: Upon Request Only
  • Duration: 2 Weeks
  • Team Structure: Solo Programming
  • Test Coverage: N/A
  • Difficulty: 7/10

This project was quite cool to work on, as raytracing is going to become more popular as a graphics option in the coming years. Additionally, it was quite difficult to implement as the intersection formulas for the spheres did not work properly for a long time. The clip shows the execution of an earlier build that was not capable of shadow effects, which were added later.
