Cat and Dog image classifier

This project was created using Jupyter Notebook.

  • Language: Python
  • Source: Jupyter Notebook File
  • Duration: 2 Weeks
  • Team Structure:Final Project for Neural Networks and Machine Learning
  • Documentation: Full Report
  • Difficulty: 7/10

For this project I used the keras python package which allowed me to create a rather advanced network in a very short ammount of time. This network achieves 91% classification accuracy on 10,000 test images. The network consists of three convolutional layers, each with max pooling, and then a densely connected layer which does the classification.
