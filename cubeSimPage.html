About

This is a personal project that I started undertaking during my final year of schooling for fun and to make something that I felt motivated to do. Before this project, I had little to no knowledge of 3D graphics or OpenGL. Now, I have a very solid beginner knowledge of how a graphics pipeline works, how objects are rotated in 3D space, how and when data in the graphics buffer should be modified, and a whole host of other things. I initally started this project to gain expirience using C++ and I have tried to keep the repository clean and follow good coding practice. I finally tagged my first release late December 2020, with functional cube rotation and camera movement.

Rubik's Cube Simulator

This project was developed and debugged in Microsoft Visual Studio.
